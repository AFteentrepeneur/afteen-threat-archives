# Disclaimer

This repository documents research methodology and publicly available information only.

**It does not, and will never, contain:**
- Exploit code or working malware samples
- Live links to dark web marketplaces, leak sites, or forums
- Stolen credentials, leaked personal data, or other victim PII
- Step-by-step instructions that could facilitate unauthorized access or fraud

**What it does contain:**
- Case timelines and analysis built from public reporting, court documents, and official law enforcement or government statements
- Pattern analysis of attacker behavior, written for defenders
- Citations back to original public sources for every factual claim

All content is produced for educational and defensive purposes. Nothing here constitutes legal advice, and nothing here should be interpreted as an endorsement or instruction to replicate any technique described.

If you believe any file in this repository violates this policy, please open an issue immediately — it will be reviewed and removed if warranted.
